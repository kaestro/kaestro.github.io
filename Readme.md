# Use of tags

 To add tags with github.io blog, I followed the solution given from 
kakao tech blog.(https://kakao.github.io/2016/07/07/tech-blog-story/)

 If you want to add another tag, create $tagname.md and follow the format of 
test.md inside the _tags directory.
